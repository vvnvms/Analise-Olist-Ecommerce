# Análise de E-commerce — Dataset Olist
Autor: Vinicius Vecchiotti
[LinkedIn](https://www.linkedin.com/in/viniciusvecchiotti/)

Análise de dados reais de e-commerce brasileiro com foco em insights de negócios. (2016-2018).

## Ferramentas
- Python (Google Colab) — EDA e visualizações
- SQL Server — queries analíticas  
- Power BI — dashboard interativo

## Análises realizadas
- Crescimento temporal de vendas
- Concentração geográfica (Pareto)
- Categorias mais vendidas
- Satisfação dos clientes
- Matriz de correlação logística
- Tempo de entrega por estado
- Segmentação RFM de clientes
- Mapa de calor geográfico de atrasos

## Dataset
[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
